# Content Guidelines

Useful Little Times content should feel fast, trustworthy, and low-pressure.

## Learning Items

Each learning item needs:

- concise title
- short fact or insight
- short explanation
- category
- card type
- at least one real source URL

Good sources include official documentation, universities, public institutions, reputable encyclopedias, museums, and scientific or educational organizations.

Avoid:

- fake URLs
- vague source labels without links
- trivia that needs too much context
- medical, legal, or financial advice
- facts that are likely to become stale quickly

## Activity Cards

Activity cards should be short, safe, and low intensity.

Each activity needs:

- home version
- office-friendly version
- remote-work version when useful
- a brief reason
- a safety note

Users should always be able to skip an activity card and see learning instead.
